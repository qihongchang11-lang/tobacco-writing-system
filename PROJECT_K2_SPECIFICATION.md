# 东方烟草报风格改写系统 - K2项目规格文档

## 🎯 项目核心目标

**打造业界领先的《新华财经》风格智能改写系统**，将任意烟草行业文稿改写为具有《新华财经》特色、兼具技术专业性与文学感染力的顶级财经报道。

### 成功标准
- **标题诗意度**：≥80%（专业编辑评估）
- **导语感染力**：≥85%（情感色彩分析）
- **修辞密度**：≥3种/篇（修辞手法统计）
- **文学一致性**：≥90%（风格相似度评估）
- **行业标杆性**：可作为行业文学化报道参考范例

## 📋 项目背景与需求

### 现状问题
当前改写系统存在**巨大质量差距**：
1. **标题过于技术直白** vs **目标诗意文学化**
2. **导语平铺直叙** vs **目标场景诗意化**
3. **结构简单直接** vs **目标层层递进文学化**
4. **语言技术说明** vs **目标散文修辞化**

### 核心挑战
- **样本库质量不足**：现有样本缺乏文学修辞元素
- **风格学习不深入**：未覆盖修辞手法、情感色彩等文学维度
- **Prompt设计技术化**：缺乏文学化写作指导
- **质量评估表面化**：未评估文学感染力和诗意表达

## 🏗️ 技术架构设计

### 核心架构：混合智能风格迁移 (Hybrid Intelligent Style Transfer)
```
三层架构：
├── 第一层：深度语义学习 (多维度向量提取)
├── 第二层：模板原型指导 (蒸馏句型+Few-shot)
└── 第三层：质量控制系统 (多维度文学评估)
```

### 关键组件
1. **IntelligentRetriever** - 混合检索(BM25+语义)
2. **XHFStyleInjector** - 文学化风格注入
3. **XHFQualityChecker** - 多维度质量评估
4. **ConstraintDecoder** - 事实信息保护

## 📊 技术规格要求

### 输入规格
- **文本类型**：烟草行业工作汇报、技术总结、管理报告
- **文本长度**：500-2000字
- **语言**：中文
- **格式**：纯文本或Markdown

### 输出规格
- **标题**：15-30字，诗意化，可引用古诗词
- **导语**：60-120字，场景化+时代背景
- **正文**：5段左右，背景→理念→实践→展望结构
- **风格**：散文式，大量修辞，文学感染力
- **质量**：达到《新华财经》发表标准

### 性能指标
- **响应时间**：≤25秒
- **成功率**：≥95%
- **并发支持**：10个并发请求
- **可用性**：99.9%

## 🎨 文学化质量标准

### 标题层标准
- **诗意化**：可引用古诗词或营造意境
- **地理化**：结合地理特征（如"金山叠翠，京口潮涌"）
- **时代化**：体现时代背景和主旋律

### 导语层标准
- **场景化**：用文学语言描绘时代背景
- **情感化**：传递积极向上的情感色彩
- **专业化**：体现行业专业性和权威性

### 正文层标准
- **散文化**：运用散文式写作手法
- **修辞化**：大量使用比喻、拟人、排比等修辞
- **韵律化**：注意语言的节奏感和韵律美
