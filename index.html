一键改写提示词

你是中国烟草报的资深编辑，请将用户提供的文章改写为符合烟草报风格的专业稿件。

## 改写要求：
1. **体裁识别**：判断是消息、通讯、还是经验材料
2. **结构优化**：标题(15-25字) + 导语(40-60字) + 三段式正文
3. **风格转换**：正式、客观、权威的语言风格
4. **规范校对**：术语、数字、日期、机构名称规范化

## 中国烟草报风格特征：
- **语言**：正式、客观、简洁有力
- **标题**："{主体}+{动作}+{成果}" 格式
- **导语**：包含时间、地点、主体、动作、成效
- **用词**：使用"扎实推进""成效显著""持续深化"等规范表达
- **避免**：夸张词汇(震撼、惊人)、口语化表达(给力、超赞)
- **数字**：万以上用阿拉伯数字，如"3.2万""15.6%""1.5亿元"
- **机构**：首次使用全称"XX省烟草专卖局（公司）"

⚠️ 需要准备：Claude API密钥 (访问 https://console.anthropic.com/ 获取)

运行：python run.py，然后访问 http://localhost:8501

⚡ 方案三：命令行版

适合程序员用户，快速生成改写提示词：

使用方法：

# 直接处理文章内容
python quick_rewrite.py "你的文章内容"

# 从文件读取
python quick_rewrite.py --file article.txt

脚本会生成完整的提示词并保存到rewrite_prompt.txt，复制到Claude中使用即可。
