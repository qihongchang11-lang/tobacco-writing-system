# 东方烟草报样本分析报告（Phase 2）

> 样本来源：用户提供的《东方烟草报》真实稿件整理版（含杭州/宁波卷烟厂、行业专题与新闻等）。  
> 主要栏目覆盖：要闻/人物与事迹、技术与设备实践、经济运行/产能优化、数字化/智能制造、政策与协同、历史文化专题等。  
> 结论用于指导改写提示词与后处理规则。  

## 1. 通用风格规律
- **语气**：官方、庄重、务实；避免夸张与感叹句。
- **句式**：常以"时间 + 地点 + 主体 + 动作/目的 + 结果/意义"起手；正文常用"背景→举措→成效→展望/下一步"。
- **术语**：贯彻、部署、推进、落实、协同、赋能、降本增效、提质增效、智能制造、数字化转型、质量管控等高频词。
- **标点**：中文全角；冒号"："用于"主题词/数字前置 + 主述"。

## 2. 栏目/类型特征对照
### 2.1 要闻/会议报道（含人物事迹）
- **标题**：主体 + 动作/主题（不使用感叹号）。  
  例：`浙江中烟杭州卷烟厂应用超高速设备推动产能优化`
- **导语**：时间地点主体 + 会议/活动/表彰 + 目的/意义（40–80字）。
- **正文**：背景（战略/问题）→组织举措（机制/团队/方法）→效果（量化指标/荣誉）→后续计划。

### 2.2 技术与设备实践（设备调试、3D打印、维保攻关）
- **标题**：技术要点/成果导向；可用"数字/结果前置 + 主题"。  
  例：`成本不到2元，开发周期缩短90%，3D打印解难题`
- **导语**：问题场景 + 技术改进 + 量化结果。
- **正文**：问题定位 → 技术/结构优化点（参数、部件、材料） → 复盘与推广价值。

### 2.3 经济运行/产能优化/经营数据
- **标题**：**强建议数字前置**，如"45.2万箱：…"。  
- **导语**：核心指标 + 同比/环比 + 结论/趋势（40–80字）。
- **正文**：数据概览 → 结构/举措 → 效益与风险 → 后续目标。

### 2.4 数字化/智能制造/质量治理
- **标题**：主题词 + 目标或路径（"以…绘就/构建…"）。  
- **导语**：转型背景 + 目标导向 + 方法抓手。
- **正文**：平台/模型/工具 → 场景成效（数据）→ 组织协同与人才 → 可复制经验。

### 2.5 政策与协同（跨域合作/机制建设）
- **标题**：两段式结构常见："主命题 + 目标/抓手"（"聚势协同谋远 跨域融合共赢——…"）。
- **导语**：政策/协同背景 + 核心机制 + 预期成效。
- **正文**：战略穿透/要素统筹/区域联动/人才共育等维度展开 + 组织与机制细节 + 结果。

### 2.6 历史文化与精神传承
- **标题**：叙事性更强；避免情绪化词汇。
- **导语**：历史场景/事件概述 + 精神内核。
- **正文**：史实叙述 → 关键节点 → 精神/价值连接当下。

## 3. 语言与数据规范
- **数字**：严格保留原始数值与单位（箱、亿元、% 等），支持小数一位；**不得新增数字**。
- **机构名**：白名单必须原样保留（国家烟草专卖局、中国烟草总公司、浙江中烟、杭州卷烟厂、宁波卷烟厂等）。
- **术语偏好**：优先使用行业内通用表达，如"质量管控""数字化转型""协同联动""智能排产"等。
- **导语**：40–80字；尽量包含时间、地点、主体、行动、结果四要素。

## 4. 提示词（Prompt）设计要点
- 按栏目注入"标题模式 + 导语结构 + 正文框架 + 术语表 + 数据/机构锁定说明"。
- 强化"不得新增数字/机构名"的约束与验证提示。
- 严格使用中文全角标点与正式语气。

## 5. 参考样本（用于BM25与风格学习）
- 杭州卷烟厂/人物事迹、设备攻关、3D打印、产能优化、质量治理；  
- 宁波卷烟厂/人机协同、数字化转型、历史专题；  
- 跨域协同与机制建设专题等。
